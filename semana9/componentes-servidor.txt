Sistema operativo:
Es el software principal del computador donde se ejecuta el servidor. Se encarga de administrar los recursos del sistema y permitir que las aplicaciones funcionen correctamente.

Node.js:
Es el entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Gracias a Node.js es posible crear y ejecutar servidores web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas disponibles y la forma en que el servidor responde a las solicitudes de los usuarios.

Puerto:
Es el punto de comunicación por donde el servidor recibe las solicitudes de los clientes. En esta práctica se utiliza el puerto 3000 para acceder al servidor.

Logs:
Son los registros que almacenan información sobre la ejecución del servidor, como mensajes, eventos y posibles errores. Ayudan a monitorear el funcionamiento de la aplicación.
